## 🔄 转换为 PDF

### 方法 1：使用 PowerPoint
1. 打开 `investment-mindset.pptx`
2. 文件 → 导出 → 创建 PDF/XPS

### 方法 2：使用 macOS Keynote
1. 用 Keynote 打开 PPTX 文件
2. 文件 → 导出为 → PDF

### 方法 3：使用 Google Slides
1. 上传到 Google Drive
2. 用 Google Slides 打开
3. 文件 → 下载 → PDF

### 方法 4：在线转换
使用免费在线工具如：
- smallpdf.com
- ilovepdf.com
- zamzar.com
